Elixir is a functional language used to build applications at large scales. It has advanced error handling and supports many concurrent actions.

Applications are broken into processes. An application can easily handle thousands of processes at once. This makes it great for projects with large amounts of data to transfer.

Elixir code is primarily made of functions. This lessens the need for state management. Data is immutable and cannot be changed.

Elixir uses pattern matching for conditional statements. This allows it to find the closest "match" and execute the given code.
